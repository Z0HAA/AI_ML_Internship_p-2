# AI/ML Internship Projects

## 📁 Projects

| Task | Project | Description |
|------|---------|-------------|
| Task 1 | [News Topic Classifier (BERT)](./bert-news-classifier/) | Fine-tuned BERT to classify news headlines into 4 categories — 92.15% accuracy. Live demo on HF Spaces. |
| Task 2 | [Telco Customer Churn Prediction](./Task_2_Telco_Churn_Pipeline/) | End-to-end Scikit-learn pipeline to predict customer churn using Logistic Regression & Random Forest. |
| Task 3 | [Housing Price Prediction](./Housing_Price_Prediction/) | Multimodal deep learning model combining house images (MobileNetV2) and tabular data for price prediction. |
